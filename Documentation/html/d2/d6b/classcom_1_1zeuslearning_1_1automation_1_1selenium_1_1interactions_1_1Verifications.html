Compare 2 strings (case-sensitive)

Compares two strings lexicographically. The comparison is based on the Unicode value of each character in the strings.

Parameters
string1{String}
string2{String}
Returns
{int} - case-sensitive comparison. returns positive integer, if string1 lexicographically precedes string2. returns negative integer, if string1 lexicographically follows string2. returns 0, if both the strings are identical. returns the difference between the lengths of string1 and string2 (string1 - string2), If there is no index position at which they differ.
